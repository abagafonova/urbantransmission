2. Какие данные обрабатываются и как

Сайт не собирает персональные данные самостоятельно. На сайте отсутствуют регистрация, личный кабинет, формы сбора данных и файлы cookie, устанавливаемые оператором.

Тем не менее при посещении сайта могут обрабатываться технические данные третьими сторонами:

2.1. Хостинг-провайдер (GitHub Pages)

Сайт размещён на платформе GitHub Pages (GitHub, Inc., США). GitHub автоматически сохраняет серверные журналы (логи), которые могут содержать IP-адрес посетителя, дату и время запроса, тип браузера. Обработка осуществляется GitHub в соответствии с его политикой конфиденциальности. Оператор настоящего сайта не имеет доступа к этим журналам и не управляет ими.

2.2. Шрифты (самостоятельный хостинг)

Шрифты, используемые на сайте, размещены непосредственно на сервере хостинга (GitHub Pages) и загружаются без обращения к внешним сервисам. Запросы к серверам Google Fonts не осуществляются; передачи IP-адреса посетителя третьим сторонам в связи со шрифтами не происходит.

2.3. Добровольное обращение по электронной почте

Если вы самостоятельно отправляете письмо на адрес a.b.agafonova@gmail.com, вы передаёте свои контактные данные (электронный адрес, содержание письма) по собственной инициативе. Эти данные используются исключительно для ответа на ваше обращение и не передаются третьим лицам. Хранение осуществляется в сервисе Gmail (Google LLC, США).

3. Цели обработки

  • Обеспечение функционирования сайта (технические логи хостинга).
  • Ответы на добровольные обращения посетителей по электронной почте.

4. Передача данных третьим лицам

Оператор не продаёт, не передаёт и не раскрывает персональные данные посетителей коммерческим третьим лицам. Технические данные обрабатываются инфраструктурными провайдерами (GitHub, Google) в рамках их собственных политик.

5. Права субъектов персональных данных

В соответствии с 152-ФЗ и GDPR вы вправе:

  • получить информацию об обрабатываемых данных;
  • потребовать уточнения или удаления данных;
  • отозвать согласие на обработку.

Для реализации прав обращайтесь по адресу: a.b.agafonova@gmail.com. По данным, которые хранятся у GitHub или Google, следует обращаться непосредственно к этим операторам.

6. Возраст пользователей

Сайт является академическим исследовательским ресурсом и не направлен на обработку данных лиц младше 18 лет. Если вам не исполнилось 18 лет, пожалуйста, не отправляйте персональные данные через электронную почту.

7. Изменения Политики

Актуальная версия Политики всегда доступна по адресу urbantransmission.org/personal-data.html. Дата последней редакции указана в начале документа.

8. Уведомление Роскомнадзора

В связи с тем, что оператор не осуществляет систематическую обработку персональных данных граждан РФ и не ведёт базы данных субъектов, уведомление РКН в соответствии с ч. 2 ст. 22 152-ФЗ не требуется (применяются исключения: данные не используются для продвижения товаров и услуг, не обрабатываются автоматизированно в базах данных, обработка ограничена разовыми обращениями по почте). При изменении характера обработки уведомление будет направлено в установленном порядке.

Who we are

This website (urbantransmission.org) is a personal academic research platform operated by Anna B. Agafonova, independent researcher. It is not a commercial service and does not collect personal data for commercial purposes.

What data is collected and why

We do not use analytics, cookies, or tracking scripts of our own. We do not set cookies, run advertising, or sell data.

The fonts used on this site are self-hosted via GitHub Pages and are served directly — no requests are made to Google Fonts or any other external font service. Your browser does not contact third-party servers to load fonts.

The site is hosted on GitHub Pages (GitHub, Inc.). GitHub may collect server access logs including IP addresses as part of standard web hosting infrastructure. See GitHub's Privacy Statement.

Contact data

If you contact the researcher by email (a.b.agafonova@gmail.com), your email address and message content will be used solely to respond to your enquiry. No email data is shared with third parties.
