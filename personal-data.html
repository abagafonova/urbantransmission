2. Какие данные обрабатываются и как

Сайт не собирает персональные данные самостоятельно. На сайте отсутствуют регистрация, личный кабинет, формы сбора данных и файлы cookie, устанавливаемые оператором.

Тем не менее при посещении сайта могут обрабатываться технические данные третьими сторонами:

2.1. Хостинг-провайдер (GitHub Pages)

Сайт размещён на платформе GitHub Pages (GitHub, Inc., США). GitHub автоматически сохраняет серверные журналы (логи), которые могут содержать IP-адрес посетителя, дату и время запроса, тип браузера. Обработка осуществляется GitHub в соответствии с его политикой конфиденциальности. Оператор настоящего сайта не имеет доступа к этим журналам и не управляет ими.

2.2. Шрифты Google Fonts

Для отображения шрифтов сайт обращается к серверам Google Fonts (Google LLC, США). При загрузке шрифта ваш браузер отправляет запрос на серверы Google, включающий IP-адрес. Это техническое взаимодействие осуществляется в рамках Политики конфиденциальности Google. Оператор настоящего сайта не получает и не хранит эти данные.

2.3. Добровольное обращение по электронной почте

Если вы самостоятельно отправляете письмо на адрес a.b.agafonova@gmail.com, вы передаёте свои контактные данные (электронный адрес, содержание письма) по собственной инициативе. Эти данные используются исключительно для ответа на ваше обращение и не передаются третьим лицам. Хранение осуществляется в сервисе Gmail (Google LLC, США).
